# StudyBud

StudyBud is a modern discussion and collaboration platform developed using Django. It enables users to create topic-based rooms, participate in conversations, and connect with people who share similar interests.

## Features

- User Registration and Login
- Create, Update, and Delete Study Rooms
- Browse Rooms by Topics
- Join Discussions in Real Time
- Search Rooms and Topics
- User Profile Management
- Recent Activity Feed
- Responsive and User-Friendly Interface
- Django Admin Panel for Management

## Tech Stack

- Django
- HTML5
- CSS3
- JavaScript
- SQLite
- Django Templates

## Installation

### Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/StudyBud.git
cd StudyBud
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

**Windows**

```bash
venv\Scripts\activate
```

**Linux / macOS**

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Apply Migrations

```bash
python manage.py migrate
```

### Run the Development Server

```bash
python manage.py runserver
```

Open your browser and visit:

```
http://127.0.0.1:8000/
```
